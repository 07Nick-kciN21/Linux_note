# IPtables
預設沒有任何規則  

## iptables -A INPUT s 192.168.1.0/24 -j DROP
在3號主機執行，只要是來自192.168.1.0/24的動作都會擋住

1號主機想ping3號會被擋住
[](../../pic/iptables1.png)

## iptables -A INPUT -s 192.168.1.1/32 -p tcp --dport 80 --j DROP
針對單一主機的特定協定進行阻擋
1號主機可以ping到3號主機，但curl卻會被阻擋
[](../../pic/input.png)

## iptables -A OUTPUT -p tcp --dport 80 -j DROP
