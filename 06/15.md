## date指令
```
[root@localhost user]# date
Tue Jun 15 21:59:14 CST 2021
```
加參數可以自己想要的格式呈現
### 常用格式
%Y：年份  
%m：月份  
%d：日  
%H：時  
%M：分  
%S：秒  
```
[root@localhost user]# date +%Y%m%d
20210615
```
```
[root@localhost user]# date +%Y/%m/%d
2021/06/15
```

## `符號
當一個指令中有一段指令被``包住  
這段指令就會先執行  
touch `date +%Y%m%d`實際執行是 touch 20210615  
```
[root@localhost a]# touch `date +%Y%m%d`
[root@localhost a]# ls
20210615
```
